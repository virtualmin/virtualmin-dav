Ενεργοποίηση συνδέσεων DAV;
Όταν επιλεγεί αυτή η δυνατότητα, οι χρήστες αλληλογραφίας και FTP σε αυτόν τον εικονικό διακομιστή θα μπορούν να χρησιμοποιούν ένα πρόγραμμα-πελάτη DAV (όπως τα Windows Network Places) για πρόσβαση στη σελίδα /dav URL και διαχείριση αρχείων ιστοσελίδας. Μόνο οι χρήστες για τους οποίους έχει ενεργοποιηθεί η πρόσβαση DAV θα μπορούν να συνδεθούν.
