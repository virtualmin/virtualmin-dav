Activer les connexions DAV?
Lorsque cette fonction est sélectionnée, les utilisateurs de messagerie et FTP au sein de ce serveur virtuel pourront utiliser un client DAV (tel que Windows Network Places) pour accéder à la page URL /dav et gérer les fichiers de page Web. Cependant, seuls les utilisateurs pour lesquels l'accès DAV a été activé pourront se connecter.
