Habiliteu la sessió de DAV?
Quan s'ha seleccionat aquesta característica, els usuaris de correu electrònic i FTP d'aquest servidor virtual podran utilitzar un client DAV (com ara Windows Network Places) per accedir a la pàgina d'URL /dav i gestionar fitxers de pàgines web. Tanmateix, només podran iniciar la sessió els usuaris als quals s’hagi habilitat l’accés DAV.
