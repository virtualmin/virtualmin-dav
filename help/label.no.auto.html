Vil du aktivere DAV-pålogginger?
Når denne funksjonen er valgt, vil e-post- og FTP-brukere på denne virtuelle serveren kunne bruke en DAV-klient (for eksempel Windows Network Places) for å få tilgang til /dav URL-siden og administrere websidefiler. Bare brukere som DAV-tilgang er aktivert, kan imidlertid logge på.
