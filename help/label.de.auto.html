DAV-Anmeldungen aktivieren?
Wenn diese Funktion ausgewählt ist, können E-Mail- und FTP-Benutzer auf diesem virtuellen Server einen DAV-Client (z. B. Windows Network Places) verwenden, um auf die URL-Seite /dav zuzugreifen und Webseitendateien zu verwalten. Nur Benutzer, für die der DAV-Zugriff aktiviert wurde, können sich jedoch anmelden.
