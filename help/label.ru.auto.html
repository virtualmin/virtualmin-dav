Включить логины DAV?
Когда эта функция выбрана, пользователи почты и FTP на этом виртуальном сервере смогут использовать клиент DAV (например, Windows Network Places) для доступа по URL /dav и управления файлами веб-страниц. Только пользователи, которым доступ к DAV был разрешен, смогут войти в систему.
