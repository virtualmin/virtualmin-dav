¿Habilitar inicios de sesión DAV?
Cuando se selecciona esta función, los usuarios de correo y FTP dentro de este servidor virtual podrán usar un cliente DAV (como Windows Network Places) para acceder a la página URL /dav y administrar los archivos de la página web. Sin embargo, solo los usuarios para los que se ha habilitado el acceso DAV podrán iniciar sesión.
