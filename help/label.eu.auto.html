DAV sarbideak gaitu?
Ezaugarri hau hautatuta dagoenean, zerbitzari birtual honen barruan mailak eta FTP erabiltzaileek DAV bezero bat (Windows Network Places, esaterako) erabili ahal izango dute /dav URL orrian sartzeko eta web orrialdeko fitxategiak kudeatzeko. DAV sarbidea gaituta duten erabiltzaileek bakarrik izango dute saioa hasteko aukera.
