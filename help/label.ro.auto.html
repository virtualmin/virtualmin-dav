Activați autentificările DAV?
Când această caracteristică este selectată, utilizatorii de mail și FTP din acest server virtual vor putea utiliza un client DAV (cum ar fi Windows Network Places) pentru a accesa pagina /dav URL și pentru a gestiona fișierele paginii web. Numai utilizatorii pentru care a fost activat accesul DAV se vor putea conecta totuși.
