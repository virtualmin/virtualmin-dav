Ativar logins DAV?
Quando esse recurso é selecionado, os usuários de email e FTP desse servidor virtual poderão usar um cliente DAV (como Locais de Rede do Windows) para acessar a página de URL /dav e gerenciar arquivos de página da web. Somente usuários para os quais o acesso DAV foi ativado poderão fazer login.
