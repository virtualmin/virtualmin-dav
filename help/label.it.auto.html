Abilitare gli accessi DAV?
Quando questa funzione è selezionata, gli utenti di posta e FTP all'interno di questo server virtuale saranno in grado di utilizzare un client DAV (come Windows Network Places) per accedere alla pagina URL /dav e gestire i file della pagina web. Tuttavia, solo gli utenti per i quali è stato abilitato l'accesso DAV potranno accedere.
